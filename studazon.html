Website Design

Project Objective

To create a web application where college students can buy or sell their textbooks to other college students

Background

During my third year at Harrisburg University of Science and Technology, I took an elective called CISC 491 Software Development Processes and Quality where I learned about agile methodologies and the software methodologies used in the professional field. For the course, I collaborated with three other computer science students on a semester-long project that entailed creating a website that utilized a database. Using this project, we applied various software development methodologies that we learn in class. For example, we determined the user stories and use cases for our project, and we prioritized and divided the tasks. My group decided to create a college textbook selling platform because we were seniors and had the problem of having multiple unneeded textbooks.

My Role

Alongside another student, I was responsible for developing the front-end using HTML, CSS, and Java. I designed and developed the login, register, and forget your password pages. I also designed the modals for the create and modify listings features. Additionally, I worked with the entire group to plan iterations and brainstorm features for the application.

Personal Goals

  • To improve my website development skills
  • To learn about the software development process
  • To apply the software development methodologies I learned in class
  • To create a project that I can continue after class
